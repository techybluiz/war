# War — Desafio de Programação em C

## Objetivos
- Cadastrar territórios com `struct` (nome, cor, tropas) usando um vetor.  
- Implementar lógica de ataque entre territórios usando ponteiros e alocação dinâmica (`malloc`, `free`).  
- Implementar sistema de missões estratégicas com verificação de condição de vitória usando alocação dinâmica e passagem por referência.

---
##  Tecnologias
- Linguagem C  
- Alocação dinâmica (`malloc`, `calloc`, `free`)  
- Vetores de `struct` e ponteiros
